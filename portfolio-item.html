Using a 7SD

Intro

In this electrical project, I employed a 7-segment display to sequentially cycle through 9 randomly generated digits. The project combines principles from digital logic design and sequential logic analysis to create a cycle of states based on specific logic conditions. To optimize the logic, I developed three different circuit designs. These designs were then implemented using NI MultiSim for virtual simulation and later physically realized using a breadboard, flip-flops, and logic gates.
